## Research Phase

### Step 3: Research the Experience

**Understand the lived experience:**
- [ ] Read first-person accounts (Reddit, support forums, memoirs)
- [ ] Understand common emotional patterns and struggles

- [ ] Note specific details that create authenticity

- [ ] Consult with experts (therapists, advocates, community members)
- [ ] Review trauma-informed language guidelines
- [ ] Identify professional resources to include (hotlines, support groups)
- [ ] Consider trigger warnings if needed

## Writing Phase

### Step 5: Write Title (6-8 words)

**Formula:** [Situation] + [Emotional Impact/Conflict]

**Guidelines:**
- [ ] 6-8 words
- [ ] Human-centered, relatable language
- [ ] Avoid jargon or clinical terms
- [ ] No judgment or blame
- [ ] Specific enough to identify the scenario

**Draft Title:**
_______________________

**Validation:**
- [ ] Word count: 6-8 words
- [ ] Clearly describes situation
- [ ] Emotionally resonant
- [ ] No anti-patterns (too long, generic, all caps, jargon)

---

### Step 6: Write Description (150-250 words)

**Formula:** [Situation] + [Conflict] + [Emotional impact]

**Guidelines:**
- [ ] 150-250 words
- [ ] Second-person perspective ("You...")
- [ ] 1-2 paragraphs
- [ ] Specific, concrete details
- [ ] Names the emotional complexity
- [ ] Validates the struggle without solving it

**Structure:**
```
Paragraph 1: Describe the situation and immediate conflict
Paragraph 2: Explore emotional impact and inner turmoil
```

**Draft Description:**
_______________________
_______________________

**Validation:**
- [ ] Word count: 150-250 words
- [ ] Second-person throughout
- [ ] Includes specific, authentic details
- [ ] Names complex emotions
- [ ] Doesn't solve the problem in description
- [ ] Resonates emotionally

---

### Step 7: Write Heart Response (100-150 words)

**Formula:** [Immediate emotional reactions] + [Validation] + [Why this is limiting]

**Guidelines:**
- [ ] 100-150 words
- [ ] Names emotional impulses without shame
- [ ] Validates why this response makes sense
- [ ] Gently shows how acting from pure emotion is limiting
- [ ] No judgment of the person

**Template:**
> Your heart wants [immediate relief]. You may feel tempted to [avoidance], [impulsive action], or [extreme reaction]. This emotional response is completely [natural/understandable] - [validate feeling]. However, acting purely from this place prioritizes [short-term comfort] over [long-term wisdom].

**Draft Heart Response:**

_______________________

**Validation:**
- [ ] Word count: 100-150 words
- [ ] Names specific emotional impulses
- [ ] Validates the feeling deeply
- [ ] Explains why impulse makes sense
- [ ] Shows limitation without shame
- [ ] No prescriptive "you should feel" language

### Step 8: Write Duty Response (100-150 words)

**Formula:** [Dharmic responsibility] + [Gita principle] + [Balanced path] + [Long-term vision]

- [ ] Balanced middle path (not extremes)
- [ ] Empowering, not preachy

**Template:**
> Your duty is to [dharmic responsibility]. The Gita teaches that [relevant principle]. This means [practical application]. The dharmic path is not [extreme A] or [extreme B], but [balanced way]. By [suggested approach], you honor both [value 1] and [value 2].

### Step 10: Write Action Steps (Exactly 5 steps, 8-12 words each)

- [ ] Practical and achievable

## Quality Validation Phase

### Step 12: Run Quality Checklist

**Content Completeness:**
- [ ] Title is 6-8 words, human-centered
- [ ] Description is 150-250 words, second-person
- [ ] Heart response is 100-150 words, validates emotions
- [ ] Duty response is 100-150 words, offers dharmic path
- [ ] Gita wisdom includes verse citation + text + interpretation
- [ ] Exactly 5 action steps, 8-12 words each
- [ ] Category assigned
- [ ] Chapter assigned (1-18)
- [ ] 3-5 tags assigned
- [ ] sc_verse field populated
- [ ] sc_verse_number field populated

**Language Quality:**
- [ ] NO robotic AI phrases in action steps
- [ ] Empathetic without patronizing
- [ ] Specific and concrete (not vague)
- [ ] Active voice throughout
- [ ] Natural, conversational tone
- [ ] No jargon or clinical language in user-facing text

**Accuracy:**
- [ ] Gita verse citation verified in authoritative source
- [ ] Verse text is accurate (not paraphrased)
- [ ] Chapter theme matches scenario content
- [ ] No made-up spiritual concepts
- [ ] Factual accuracy of situation description

**Sensitivity (if applicable):**
- [ ] Crisis scenarios include professional resources
- [ ] Trauma-informed language used
- [ ] No toxic positivity or spiritual bypassing
- [ ] Acknowledges when professional help is needed
- [ ] Trigger warnings if appropriate

---

### Step 13: Anti-Pattern Check

**Review Against Anti-Patterns:**

**Title:**
- [ ] NOT too long (over 12 words)
- [ ] NOT too generic
- [ ] NOT all caps
- [ ] NOT using jargon
- [ ] NOT judgmental

**Description:**
- [ ] NOT too brief (under 100 words)
- [ ] NOT in third-person
- [ ] NOT solving the problem
- [ ] NOT using clinical language
- [ ] NOT minimizing the struggle

**Heart Response:**
- [ ] NOT shaming the emotion
- [ ] NOT skipping validation
- [ ] NOT prescriptive about feelings
- [ ] NOT too brief (under 75 words)

**Duty Response:**
- [ ] NOT imposing impossible standards
- [ ] NOT ignoring real constraints
- [ ] NOT being preachy
- [ ] NOT vague ("just have faith")
- [ ] NOT all-or-nothing thinking

**Gita Wisdom:**
- [ ] NO made-up verses
- [ ] NO missing verse text
- [ ] NOT forcing verses that don't fit
- [ ] NOT skipping interpretation
- [ ] NOT generic without specifics

**Action Steps:**
- [ ] NO robotic AI phrases
- [ ] NO vague actions
- [ ] NOT too many steps (over 6)
- [ ] NOT too few (under 4)
- [ ] NO passive voice
- [ ] NO steps requiring unavailable resources

---

### Step 14: Peer Review (If Possible)

**Internal Review:**
- [ ] Read aloud to check flow and tone
- [ ] Check from user perspective: Is this helpful?
- [ ] Verify emotional resonance: Does this validate the struggle?
- [ ] Confirm practical value: Are steps actually doable?

**External Review (for crisis scenarios):**
- [ ] Mental health professional review (if mental health crisis)
- [ ] Community advocate review (if marginalized experience)
- [ ] Cultural sensitivity review (if culture-specific)
- [ ] Accuracy review by someone with lived experience

**Feedback Integration:**
- [ ] Address all critical feedback
- [ ] Consider all suggestions
- [ ] Document changes made

## Final Review & Publishing

### Step 17: Final Quality Check

**Read through entire scenario one more time:**
- [ ] Does this honor the human struggle?
- [ ] Does this apply Gita wisdom authentically?
- [ ] Will this help someone in genuine distress?

- [ ] Are action steps actually actionable?

**A scenario is ready to publish when:**

1. ✅ All fields are complete and accurate
2. ✅ Word counts are within target ranges
3. ✅ NO robotic AI language anywhere
4. ✅ Gita verse verified and accurately applied
5. ✅ Empathetic, practical, and empowering
6. ✅ Passes all quality validation checks
7. ✅ Passes all technical validation
8. ✅ Expert review completed (if crisis scenario)
9. ✅ JSON format valid
10. ✅ Tested in staging environment
