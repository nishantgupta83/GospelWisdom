# Performance Monitoring Agents

Automated agents for iOS and Android performance monitoring.

## Usage

**iOS**:
```bash
python ios/ios_performance_agent.py
```

**Android**:
```bash
python android/android_performance_agent.py
```

## Metrics Monitored
- FPS (target: 55+)
- Memory (target: <150MB iOS, <120MB Android)
- Launch time (target: <2s)
- Battery drain
- CPU usage
